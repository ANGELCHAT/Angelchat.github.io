# Introduction

This Software Development Kit, written in Go, is a tool for developers to build external backend apps, which extend LiveChat features. The SDK makes it easy to use the [Agent Chat API](/messaging/agent-chat-api/), the [Customer Chat API](/messaging/customer-chat-api/) and the [Configuration API](/management/configuration-api/).
